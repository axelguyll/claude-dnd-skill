# Issue tracker

Tracker: **GitHub Issues** on `axelguyll/claude-dnd-skill` (origin fork), via `gh` CLI.

- `to-issues` / `triage` / `to-prd` create and read issues with `gh issue ...`.
- External PRs are **not** part of the triage queue (solo project; revisit if collaborators appear).
- Upstream (`neuralinitiative/claude-dnd-skill`) is pull-only; never file issues there.
